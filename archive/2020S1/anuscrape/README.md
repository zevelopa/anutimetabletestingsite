# anuscrape

## Python Scraper

### Python requirements:

As per Pipfile:

- Python version: 3.7
- Requests package
- BeatifulSoup4 package
- Pipenv is recommended but not required.

### Usage instructions

1. Run ./scraper.py (Takes about 2 mins from outside the ANU network, 3 mins inside)
2. Open a pull request on the github website
